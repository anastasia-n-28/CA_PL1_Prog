# Опис програмного забезпечення до Практичної роботи №1

## 1. Загальні відомості
Програма розроблена для автоматизації обчислень, передбачених індивідуальним завданням варіанта №17. Вона представляє собою веб-застосунок, що дозволяє користувачеві виконувати перетворення чисел та кодування інформації через графічний інтерфейс.

**Розробник:** Набока А.О., група ПЗПІ-24-1.  
**Призначення:** Автоматизація переведення чисел між системами числення, виконання двійкової арифметики, кодування чисел у формат IEEE 754 та тексту в ASCII.

## 2. Технічні характеристики
* **Мова програмування:** Python 3.10+
* **Фреймворк (Backend):** FastAPI — сучасний веб-фреймворк для побудови API.
* **Інтерфейс (Frontend):** HTML5, CSS3, JavaScript (без використання сторонніх бібліотек).
* **Архітектура:** Клієнт-серверна (REST API). Сервер виконує всі математичні обчислення, а браузер відповідає за відображення результатів.

## 3. Функціональні можливості
Програма реалізує 5 основних модулів:

1.  **Конвертер цілих чисел:**
    * Підтримує переведення між двійковою (2), вісімковою (8), десятковою (10) та шістнадцятковою (16) системами.
    * Приклад (Вар. 17): $183_{10} \to B7_{16}$.

2.  **Конвертер дробових чисел:**
    * Дозволяє задавати точність обчислень (кількість знаків після коми).
    * Реалізує алгоритми для роботи з дробовою частиною (множення на основу).
    * Приклад (Вар. 17): $0.563_{10} \to 0.10010_2$.

3.  **Арифметичний модуль:**
    * Виконує операції додавання, віднімання та множення.
    * Відображає результат одночасно у десятковій та двійковій системах.
    * Приклад (Вар. 17): $178 + 35 = 213$ ($11010101_2$).

4.  **Модуль IEEE 754:**
    * Конвертує дробові числа у 32-бітний формат з плаваючою комою (Single Precision).
    * Виводить бінарне представлення (Знак | Експонента | Мантиса) та HEX-код.
    * Приклад (Вар. 17): $28.25 \to 41E20000_{16}$.

5.  **ASCII Кодер:**
    * Приймає текстовий рядок (прізвище).
    * Генерує таблицю з кодами кожного символу (DEC, HEX, BIN).
    * Приклад (Вар. 17): "Naboka".

## 4. Інструкція користувача

### 4.1. Встановлення та запуск
1.  Встановіть інтерпретатор Python (версія 3.7 або вище).
2.  Встановіть необхідні бібліотеки командою в терміналі:
    ```bash
    pip install fastapi uvicorn
    ```
3.  Розмістіть файл `main.py` та папку `static` (з файлом `index.html`) в одній директорії.
4.  Запустіть сервер командою:
    ```bash
    uvicorn main:app --reload
    ```
5.  Відкрийте веб-браузер та перейдіть за адресою: [http://127.0.0.1:8000](http://127.0.0.1:8000).

### 4.2. Робота з програмою
* **Для переведення чисел:** Введіть число у відповідне поле, оберіть початкову та цільову системи числення зі списків, натисніть "Перевести".
* **Для арифметики:** Введіть два десяткових числа, оберіть операцію (+, -, *) та натисніть "Обчислити".
* **Для кодування тексту:** Введіть своє прізвище латиницею у поле блоку "ASCII Кодер" та натисніть "Закодувати".

Результати обчислень з'являються миттєво у відповідних блоках на екрані.
